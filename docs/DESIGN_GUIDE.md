## Color Palette

### Primary Colors
- **Purple**: `#7c3aed` / `#6b21a8` - Buttons, accents, stats
- **Amber**: `#fbbf24` / `#f59e0b` - CTAs, badges

### Usage
- Hero background: `slate-900` with gradient
- Page background: `#f3f4ff` (light purple-ish)
- Cards: `white` with subtle shadows
- Text: `slate-700` to `slate-900`
- Footer: `slate-900` with `slate-200` text

## Typography Hierarchy

```
Hero Title:        text-5xl (48px)  - bold
Section Headings:  text-3xl (30px)  - bold
Subsections:       text-2xl (24px)  - bold
Card Titles:       text-lg (18px)   - semibold
Body Text:         text-sm (14px)   - regular
Small Text:        text-xs (12px)   - regular
Tiny Text:         text-[11px]      - regular
Labels:            text-[10px]      - semibold, uppercase
```

---

## Spacing & Layout

- **Max Width**: 1152px (max-w-6xl)
- **Padding**: px-4 (mobile), px-6 (desktop)
- **Section Spacing**: py-12 to py-24
- **Card Gaps**: gap-4 to gap-8
- **Border Radius**: rounded-xl for cards, rounded-full for buttons

---

## Interactive Elements

### Buttons
- **Primary**: Purple background, white text, rounded-full
- **Secondary**: Amber background, dark text, rounded-full
- **Outline**: Border only, rounded-full
- **Hover**: Darker shade

### Cards
- **Shadow**: `shadow-[0_10px_25px_rgba(15,23,42,0.12)]`
- **Hover**: Increased shadow
- **Border**: None or subtle slate border

### Links
- **Default**: slate-700
- **Hover**: slate-900
- **Active**: slate-900 with background

---

## Responsive Design

### Mobile (<768px)
- Stack sections vertically
- Hide desktop navigation (can add mobile menu)
- Single column layouts
- Smaller typography
- Reduced padding

### Tablet (768px - 1024px)
- 2-column layouts where appropriate
- Medium typography
- Standard padding

### Desktop (>1024px)
- 3-4 column layouts
- Large typography
- Maximum padding
- Full navigation visible

---

## Key Design Principles

1. **Clean & Modern**: Minimal clutter, ample white space
2. **Professional**: Suitable for an alumni association
3. **Accessible**: Good contrast, readable fonts
4. **Consistent**: Unified color scheme and spacing
5. **Responsive**: Works on all devices
6. **Fast**: Optimized Next.js performance
